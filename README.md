# Wilnder

![Wilnder Screenshot](./screenshot.png)

Минималистичный swipe-app в стиле Tinder для карточек Wildberries.

## Что это

Wilnder показывает случайные WB-товары в формате карточек:
- свайп вправо — лайк
- свайп влево — скип
- вкладки для просмотра лайкнутых и скипнутых

## Быстрый старт

```bash
npm install
npm start
```

Открой: [http://localhost:3000](http://localhost:3000)

## Seed-режим

Чтобы получить повторяемую последовательность карточек:

```text
http://localhost:3000/?seed=my-seed
```

## Стек

- Node.js + Express
- Vanilla HTML/CSS/JS
